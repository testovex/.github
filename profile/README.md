## What is Testovex?

Testovex is a multi-tenant **Automation Center of Excellence (ACOE)** platform that lets QA teams build, manage, and run end-to-end tests — without writing code.

Built on **Playwright**, designed for **enterprise teams**, and governed by role-based access control across organisations, projects, and test suites.

---

## Key Features

- 🖱️ **No-Code Test Builder** — record and build tests visually using the Testovex Chrome extension
- 🤖 **AI Test Generator** — generate tests automatically from a URL *(coming soon)*
- 🔁 **Reusable Components** — shared step libraries across projects
- 🌐 **Multi-Tenant** — org-level isolation, RBAC, and variable locking
- 📊 **Analytics & Reporting** — run history, pass/fail trends, Grafana-ready metrics
- 🎥 **Run Diagnostics** — video, screenshots, and Playwright traces per test run
- 🔒 **Enterprise Security** — JWT auth, audit trails, encrypted variables, CSP headers
- ⚙️ **CLI Support** — trigger runs from CI/CD pipelines
- 🗂️ **Version History** — snapshot and restore test cases, suites, and components
